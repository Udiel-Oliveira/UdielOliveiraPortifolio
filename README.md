## 🎯 Propósito

Este repositório contém o código-fonte do meu portfólio pessoal — um espaço criado para apresentar minha trajetória como desenvolvedor Front End de forma clara, direta e profissional.

O site foi desenvolvido do zero por mim, e é em si mesmo um projeto: cada escolha de tecnologia, animação e estrutura reflete minha forma de pensar e construir interfaces.

---

## 👥 Para quem é este portfólio?

Principalmente para **recrutadores e empresas** que desejam conhecer meu perfil técnico, ver projetos reais que já desenvolvi e entender como eu trabalho antes de uma conversa.

## 🛠️ Tecnologias utilizadas

| Camada              | Tecnologia                  |
|---------------------|-----------------------------|

| Linguagem           | TypeScript                  |
| Estilização         | Tailwind CSS v4 + CSS Modules |
| Animações           | GSAP 3 (SplitText)          |
| Componentes         | Swiper, Akar Icons          |
| Deploy & Analytics  | Vercel                      |


## ✅ Diferenciais

**Design responsivo** — experiência consistente em qualquer dispositivo, do celular ao desktop.

**Animações com propósito** — GSAP usado para guiar a atenção do visitante, não para decoração.

**Conteúdo orientado a dados** — projetos, badges e certificações gerenciados via JSON, facilitando atualizações sem alterar o código.

**Performance e SEO** — metadados completos, Open Graph configurado e monitoramento de performance via Vercel Speed Insights.

**Código limpo e componentizado** — arquitetura modular com separação clara entre layout, seções e componentes de UI reutilizáveis.
